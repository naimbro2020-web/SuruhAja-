# SuruhAja! 🛵
### Jasa Serba Bisa Bandar Lampung

Website ini dibuat untuk memenuhi kebutuhan harian masyarakat Bandar Lampung, 
khususnya mahasiswa — mulai dari antar jemput, kirim barang, belanja titip, 
antar makanan, bersih-bersih kost, hingga potong rumput. Semua bisa, tinggal suruh aja!

## 🔧 Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- Template: Tooplate 2158 Forge Reality

## 📋 Fitur
- Halaman layanan lengkap
- Sistem harga transparan
- Order langsung via WhatsApp
- Peta area layanan Bandar Lampung
- Desain responsif (mobile-friendly)

## 🚀 Cara Pakai
1. Clone repo ini
2. Taruh foto di folder `images/`
3. Ganti nomor WhatsApp di `index.html`
4. Buka `index.html` di browser

## 📍 Area Layanan
Bandar Lampung & sekitarnya
